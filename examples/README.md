# Examples

Drop example files here to test TalkForge:

- `portrait.jpg` — a clear front-facing portrait (at least 256 × 256 px)
- `speech.wav` — a short speech clip (16 kHz mono WAV recommended)

These files are intentionally excluded from the repository to keep the repo
size small.  Any royalty-free portrait + audio combination will work.

## Quick test (after setup)

```bash
# From the TalkForge project root:
python main.py --share
```

Then open the printed URL, drop in your files, and click **Generate Video**.
